# Compressão de Imagens com Decomposição em Valores Singulares (SVD)

## Sobre o Repositório

Este repositório contém a implementação de um programa em Python desenvolvido no Google Colaboratory como parte de um trabalho avaliativo da disciplina de Métodos Numéricos Computacionais do curso de Bacharel em Ciência da Computação.

O objetivo do trabalho foi demonstrar a aplicação da Decomposição em Valores Singulares (SVD - Singular Value Decomposition) em imagens, evidenciando o impacto da redução dos valores singulares na qualidade da imagem resultante.
